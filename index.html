GitHub Pages’ta Yayınlama (kısa rehber)

  1. GitHub hesabınızla oturum açın veya kayıt olun.
  2. Yeni repo oluşturun: adı tam olarak username.github.io (username = GitHub kullanıcı adınız, örn. harunnalcakan.github.io).
  3. Repo Public olsun. Add file → Create new file deyip bu dosyanın adını index.html olarak kaydedin ve Commit edin (bu sayfanın tamamını yapıştırın).
  4. Repo içinde Settings → Pages bölümüne gidin. Branch olarak main ve Folder olarak / (root) seçili olsun. Save edin.
  5. Site adresiniz https://username.github.io olarak yayına alınır. Enforce HTTPS seçeneğini açın.

Hazır hissetmiyorsanız: Settings → Pages → Theme Chooser ile bir Jekyll teması seçip başlangıç sayfası oluşturabilirsiniz.

İpucu

  • Bu dosya tek başına yeterli. Daha sonra assets/ klasörü ve çoklu sayfalar ekleyebilirsiniz.
  • README.md içine kısaca sitenizi açıklayın. Lisans için LICENSE ekleyin.
  • Özel alan adı kullanacaksanız repo köküne CNAME dosyası ekleyin.
